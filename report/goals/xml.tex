\subsection{Цель \texttt{xml}}

\subsubsection{Листинг}

\begin{lstlisting}[caption={Часть содержимого файла \texttt{build.xml}},captionpos=b]
    <target name="xml">
        <echo message="Validating XML and XHTML files in the project..."/>

        <xmlvalidate lenient="true" failonerror="false">
            <fileset dir="."
                     includes="${fse.xml.includes}"
                     excludes="${fse.xml.excludes}"/>
        </xmlvalidate>

        <echo message="Done task 'xml'!" />
    </target>
\end{lstlisting}

\subsubsection{Комментарии}

Валидирует XML и XHTML файлы проекта на синтаксическую корректность. Использует \texttt{xmlvalidate} в режиме \texttt{lenient="true"}. Ошибки не останавливают сборку.

\subsubsection{Значения используемых параметров}

\begin{lstlisting}[language=Properties,caption={Часть содержимого файла \texttt{build.properties}},captionpos=b]
fse.xml.includes = **/*.xml,**/*.xhtml
fse.xml.excludes = build-test/**,build/**,target/**,alt-version/**,.ant/**,sandbox/**
\end{lstlisting}
