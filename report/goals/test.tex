<target name="compile-tests">
        <echo message="Compiling JUnit tests..."/>

        <delete dir="${fse.test.target}" />
        <mkdir dir="${fse.test.target}"/>

        <javac srcdir="${fse.test.source}"
               destdir="${fse.test.target}"
               source="${fse.compile.jvm.source}"
               target="${fse.compile.jvm.target}"
               includeantruntime="false">
            <classpath>
                <path refid="fse.ivy.path.test"/>
                <pathelement location="${fse.compile.target}"/>
            </classpath>
        </javac>

        <echo message="Done task 'compile-tests'!" />
    </target>

    <target name="test" depends="build">
        <echo message="Running JUnit tests..."/>

        <antcall target="compile-tests" />

        <condition property="junit.reports.arg" value="--reports-dir=${fse.test.target}/${fse.test.reports.subdir}" else="">
            <isset property="fse.test.report.output"/>
        </condition>

        <java classname="org.junit.platform.console.ConsoleLauncher" fork="true" failonerror="true">
            <jvmarg value="-XX:+EnableDynamicAgentLoading"/>
            <jvmarg value="-Xshare:off"/>
            <classpath>
                <path refid="fse.ivy.path.compile"/>
                <path refid="fse.ivy.path.test"/>
                <pathelement location="${fse.compile.target}"/>
                <pathelement location="${fse.test.target}"/>
            </classpath>
            <arg value="execute"/>
            <arg value="--scan-classpath"/>
            <arg value="--details=verbose"/>
            <arg line="${junit.reports.arg}"/>
        </java>

        <echo message="Done task 'test'!" />
    </target>
\end{lstlisting}

\subsubsection{Комментарии}

\texttt{compile-tests} — компиляция JUnit-тестов с использованием отдельного classpath \texttt{fse.ivy.path.test}, который содержит тестовые зависимости (JUnit, Mockito). Перед компиляцией директория с тестами удаляется для обеспечения чистой сборки.

\texttt{test} — запуск тестов через JUnit ConsoleLauncher. Перед запуском вызывает \texttt{compile-tests} через \texttt{antcall}, чтобы гарантировать актуальность тестовых классов. Поддерживает условную генерацию XML-отчётов при передаче параметра \texttt{fse.test.report.output}. В случае падения любого теста сборка останавливается с ошибкой.
