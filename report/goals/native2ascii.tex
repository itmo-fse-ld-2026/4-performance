\subsection{Цель \texttt{native2ascii}}

\subsubsection{Листинг}

\begin{lstlisting}[caption={Часть содержимого файла \texttt{build.xml}},captionpos=b]
  <target name="native2ascii">
    <echo message="Converting localization files from ${fse.i18n.native.dir} to ${fse.i18n.target.dir}..."/>

    <delete dir="${fse.i18n.target.dir}" />
    <mkdir dir="${fse.i18n.target.dir}" />

    <native2ascii encoding="${fse.i18n.encoding}"
                  src="${fse.i18n.native.dir}"
                  dest="${fse.i18n.target.dir}"
                  includes="${fse.i18n.includes}"
                  ext="${fse.i18n.ext}"/>

    <echo message="Done task 'native2ascii'!" />
  </target>
\end{lstlisting}

\subsubsection{Комментарии}

Преобразует файлы локализации из UTF-8 в ASCII-совместимый формат с escape-последовательностями. Это необходимо для корректной работы русскоязычных строк в Java-приложении независимо от платформы. Перед преобразованием директория назначения удаляется для обеспечения чистой сборки.

\subsubsection{Значения используемых параметров}

\begin{lstlisting}[language=Properties,caption={Часть содержимого файла \texttt{build.properties}},captionpos=b]
fse.i18n.native.dir = src/main/i18n-native
fse.i18n.target.dir = build-i18n
fse.i18n.includes = weblab/messages*.properties
fse.i18n.encoding = UTF-8
fse.i18n.ext = .properties
\end{lstlisting}
